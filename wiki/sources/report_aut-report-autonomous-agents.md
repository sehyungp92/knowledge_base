## Key Claims

1. Self-play training is a frozen approximation of what runtime memory should continuously do — baking accumulated experience into weights is strictly inferior to correctable inference-time memory update
2. Agent resistance to human steering is a memory failure, not an autonomy preference — agents without coherent episodic memory interpret interruption as context destruction rather than course correction
3. Current agent benchmarks systematically overestimate deployment readiness because they evaluate isolated single-agent performance, abstracting away the coordination overhead that dominates real-world 
4. Human-in-the-loop interruption should be uncertainty-gated on task-state variance and action irreversibility, not scheduled at fixed workflow stages.
