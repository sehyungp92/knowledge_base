# Delete Source

Remove a source and all its dependent data from the knowledge base.

## Usage

```
/delete <source_id_or_url>
```

Examples:
- `/delete 01KJ6N33291HGRAGQ600WAQ9XF`
- `/delete https://www.youtube.com/watch?v=abc123`

This skill is handled by a direct Python handler — no Claude CLI execution needed.
