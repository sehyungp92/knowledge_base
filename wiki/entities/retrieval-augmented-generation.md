## Overview

A technique that uses heuristic retrieval to select relevant tokens from a large corpus to include in a model's context. Manifest argues power retention makes RAG unnecessary by allowing the model to handle all information directly.

8. ReSearch is trained from scratch without any labeled data on reasoning chains. (from "ReSearch: Learning to Reason with Search for LLMs via Reinforcement Learning")
